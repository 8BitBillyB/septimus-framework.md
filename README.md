## The Seven Extractors

1. **KEYSTONE** - Extract exact language, crystallized phrases
2. **THREADBARE** - Extract concept origins, relationships
3. **BLUEPRINT** - Extract rebuildable structure, schemas
4. **CRASHTEST** - Extract failure modes, weaknesses
5. **SPARKGAP** - Extract emergent patterns, surprises
6. **ROSETTA** - Extract translations, bridges to other contexts
7. **OUROBOROS** - Extract meta-observations, recursive insights

**PRIME** - Synthesize all seven extractions into coherent output
